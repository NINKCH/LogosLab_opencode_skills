# Dashboard Prompt Example

## Primary Prompt

Modern web application dashboard design

**Visual Style:**
- Design system: Material Design 3
- Color scheme: Primary #1976D2 (blue), Secondary #424242 (dark gray), Accent #FFC107 (amber)
- Typography: Roboto font family, headings 24px/600, body 16px/400
- Visual tone: Clean, professional, data-focused

**Layout:**
- Structure: Sidebar navigation + main content area
- Sections: Top app bar, left sidebar (240px), main content (fluid), card-based layout
- Spacing: 16px base unit, 24px between cards
- Responsive: Desktop-first, collapse sidebar on mobile (<768px)

**Components:**
- Top App Bar: Logo left, search center, user avatar right, elevation 2
- Sidebar: Navigation menu with icons, active state highlighted
- Stat Cards: 4 cards in row, icon + number + label, subtle shadow
- Chart Area: Line chart showing trends, legend, tooltips
- Data Table: Sortable columns, pagination, row hover effect

**Content:**
- Placeholder data for metrics (users, revenue, growth)
- Sample chart with 6 months of data
- Table with 10 rows of user data

**Interactions:**
- Card hover: Slight elevation increase
- Button hover: Background color darkens
- Smooth transitions: 200ms ease-in-out

**Accessibility:**
- WCAG AA contrast ratios
- Clear focus indicators
- Semantic HTML structure

## Alternative Variations

### Variation 1: Dark Mode
Same as above but with dark background (#121212), light text (#FFFFFF), adjusted card colors

### Variation 2: Minimalist
Reduced shadows, more whitespace, monochrome color scheme with single accent color

## Negative Prompts
- Cluttered layout
- Inconsistent spacing
- Poor contrast
- Outdated design patterns
- Comic sans font

## Technical Parameters
- Aspect Ratio: 16:9
- Resolution: 1920x1080
- Format: PNG
- Quality: High
