# Mobile Login Screen Prompt Example

## Primary Prompt

iOS mobile app login screen design

**Visual Style:**
- Design system: iOS Human Interface Guidelines
- Color scheme: Primary #007AFF (iOS blue), Background #FFFFFF, Text #000000
- Typography: SF Pro Display, heading 28px/bold, body 17px/regular
- Visual tone: Clean, minimal, trustworthy

**Layout:**
- Structure: Vertical stack, centered content
- Sections: Logo area (top 30%), form (middle 40%), footer (bottom 30%)
- Spacing: 20px horizontal padding, 16px vertical gaps
- Responsive: iPhone 14 Pro (393x852), safe area aware

**Components:**
- Logo: Centered, 80x80px, top margin 60px
- Title: "Welcome Back", 28px bold, centered
- Input Fields: Email and password, rounded corners (12px), 50px height
- Primary Button: Full width, 50px height, rounded (25px), blue background
- Text Link: "Forgot Password?", 15px, blue color, centered
- Social Login: Google and Apple buttons, outlined style

**Content:**
- App logo placeholder
- "Welcome Back" title
- "Sign in to continue" subtitle
- Email and password placeholders
- "Sign In" button text
- "Forgot Password?" link
- "Or continue with" divider
- Social login buttons

**Interactions:**
- Input focus: Blue border, subtle shadow
- Button press: Slight scale down (0.98)
- Keyboard: Dismiss on background tap

**Accessibility:**
- Large touch targets (44x44pt minimum)
- Clear labels for screen readers
- High contrast text

## Alternative Variations

## Negative Prompts
- Cluttered interface
- Small touch targets
- Low contrast text
- Outdated iOS design
- Inconsistent spacing
