# Generation Parameters by Use Case

## Dashboard/Admin Interfaces

**Aspect Ratio**: 16:9 or 16:10
**Resolution**: 1920x1080 or higher
**Style**: Clean, professional, data-focused
**Quality**: High
**Iterations**: 4-6 variations

**Recommended Settings**:
- Midjourney: `--ar 16:9 --style raw --q 2`
- DALL-E 3: `size: 1792x1024, quality: hd`
- Stable Diffusion: `1024x768, steps: 40, cfg: 7.5`

---
